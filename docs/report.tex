\section*{Abstract}
This report evaluates the efficacy of a Siamese Transformer architecture in mapping error correction code operations into a geometric embedding space. Initial experiments demonstrated that generative decoding struggles with edit distance tasks due to shift-induced error propagation. By utilizing an encoder-only architecture trained with Triplet Margin Loss, the network successfully learns the structural relationships between sequences. The model achieves an RMSE of 2.092 and a Triplet Accuracy of 0.7238, demonstrating that attention mechanisms can effectively map discrete sequence mutations into a measurable metric space, outperforming standard recurrent and convolutional baselines in absolute error and separation accuracy. A subsequent extension replaces the metric embedding with a Connectionist Temporal Classification head, which recovers explicit bit sequences and enables direct Bit Error Rate evaluation while retaining the encoder-only design that avoids shift-induced error propagation.

\section*{Methodology and Architecture}
Standard generative sequence-to-sequence models face fundamental limitations when applied to edit distance problems. A single uncorrected insertion or deletion operation shifts the predicted sequence, causing positional loss functions to heavily penalize structurally valid outputs. To resolve this instability, the architecture was refactored into an encoder-only Siamese network designed to evaluate Euclidean distances in a continuous manifold.

The dataset utilizes a triplet schema consisting of anchor, positive (correlated), and negative (corrupted) sequences. The architecture is a Siamese Transformer featuring an embedding dimension of $d_{model} = 256$, utilizing 8 attention heads across 4 encoder layers. Mean pooling collapses the high-dimensional encoder memory into a single representative embedding vector. 

The network was trained over 30 epochs using the AdamW optimizer. Optimization was governed by a Triplet Margin Loss function, defined as:
\begin{equation}
    \mathcal{L}(a, p, n) = \max \{ d(a_i, p_i) - d(a_i, n_i) + {\rm margin}, 0 \}
\end{equation}
where $d$ is the pairwise Euclidean distance, $a_i$ is the anchor, $p_i$ is the positive sample, $n_i$ is the negative sample, and the margin is set to 1.0. To calculate scale-comparable evaluation metrics (RMSE and MAE), a 1D linear regression was fitted to project the continuous Euclidean embedding distances onto the discrete Levenshtein edit distance scale.

\section*{Empirical Results}
Performance was evaluated by calculating the distances for 10,000 sequence triplets. The network yielded an average positive distance of 2.7128 and an average negative distance of 3.3750, resulting in a Triplet Accuracy of 0.7238.

The data reveals a significant architectural tradeoff. The proposed Siamese Transformer achieves an RMSE of 2.092 and an MAE of 1.670, substantially outperforming traditional models like CGK and CNN-ED in minimizing absolute distance error. Furthermore, its Triplet Accuracy of 0.7238 proves a strong capacity for binary sequence separation. 

However, the correlation coefficients (Pearson 0.455, Spearman 0.452) trail the convolutional and alignment baselines. This indicates that while Triplet Margin Loss effectively establishes tight absolute boundaries and pushes local pairs past the defined margin, it does not inherently force a globally linear ranking across the entire severity scale. Because the architecture maps relationships rather than predicting discrete sequences, Hamming Loss is structurally inapplicable. While the specialized lab architecture maintains the highest overall fidelity, the variance captured by the Siamese encoder proves that multi-head attention mechanisms provide a highly accurate estimation of error correction metrics without the overhead of generative decoding.

\section*{Current Extension: Connectionist Temporal Classification}
The Siamese architecture quantifies the distance between two sequences but does not emit a recovered sequence, so it cannot produce a Bit Error Rate directly. To recover explicit bit sequences while preserving the shift-invariance of the encoder-only design, the architecture was extended with a Connectionist Temporal Classification head. The encoder retains the embedding dimension of $d_{model} = 256$, the 8 attention heads, and the 4 encoder layers of the prior phase. The pooling stage is removed and replaced with a linear projection onto three classes, comprising one blank symbol and the two binary code symbols.

Connectionist Temporal Classification requires that the input time dimension $T$ accommodate the target length plus the count of adjacent repeated target symbols, since at least one blank must separate consecutive identical emissions. For a binary target of length 100 with extended runs of identical symbols, this requirement can approach a time dimension of 200. To guarantee feasibility, the encoder output is temporally upsampled by a factor of two using nearest-neighbor interpolation, producing an alignment lattice of length $T = 200$ from an input of length 100. The training objective is the Connectionist Temporal Classification negative log likelihood,
\begin{equation}
    \mathcal{L}_{CTC} = - \log \sum_{\pi \in \mathcal{B}^{-1}(\mathbf{y})} \prod_{t=1}^{T} p(\pi_t \mid \mathbf{x}),
\end{equation}
where $\mathbf{y}$ is the target bit sequence, $\pi$ ranges over the alignment paths, and $\mathcal{B}$ is the many-to-one collapsing map that removes blanks and merges repeated symbols. The loss is configured with the infinite-cost guard enabled so that alignments shorter than the target contribute zero gradient rather than diverging.

Inference uses greedy decoding. The class of maximum probability is selected at each of the 200 time steps, consecutive duplicate classes are collapsed, and the blank symbol is removed, yielding a variable length recovered bit sequence. Because the recovered length rarely equals the target length, the recovered sequence is compared to the original through a length-aware Levenshtein alignment that resolves substitutions, insertions, and deletions. From the alignment the Bit Error Rate is computed as the edit distance divided by the target length, the Block Error Rate as the fraction of sequences with any residual error, and the Hamming Loss as the fraction of aligned positions in error. This decoding stage is what distinguishes the Connectionist Temporal Classification model from the Siamese phase, since it produces explicit bit sequences and therefore makes the Hamming Loss metric, previously inapplicable to the encoder-only embedding, directly computable.

The temporal upsampling and loss computation were first verified on the CPU through a single forward and backward pass, confirming finite loss and stable gradient flow. The greedy decoder and the alignment metrics were verified on synthetic sequences, reproducing the expected blank collapse behavior and exact substitution, insertion, and deletion counts. Convergence was then verified on a ten thousand sample triplet subset on the CPU, where the Connectionist Temporal Classification loss decreased monotonically across five epochs from 0.8807 to 0.7951. The full training run on the large edit distance dataset using a CUDA device, together with the resulting Bit Error Rate, Block Error Rate, and Hamming Loss against the BCJR baseline, is in progress and will be reported once convergence on the complete dataset is confirmed.
